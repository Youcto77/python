Les variables :

En python, les variables peuvent contenir 4 types de données : Chaîne de caractères, Nombre entier, Nombre flottant et Booléen

age = 17 #Nombre entier

decimal = 23.5 #Nombre flottant

prenoms = "Enzo" #Chaîne de caractères

decisions = True / False #Booléen

Comment utiliser des variables :

Pour utiliser des variables en Python, commencez par leur donner un nom et une valeur avec = (par exemple, age = 25). Vous pouvez ensuite appeler cette variable par son nom pour afficher, modifier ou manipuler sa valeur dans le code. Les variables rendent le programme plus lisible et permettent de travailler facilement avec des données sans avoir à les retaper.

prenoms = "Enzo" #Création de la variable "prenoms"


print(prenoms) #Envoi dans la console "Enzo"

print("Bonjour je suis" + prenoms + ".") #Envoi dans la console "Bonjour je suis Enzo."

print(f"Bonjour je suis {prenoms}.) #Envoi aussi dans la console "Bonjour je suis Enzo." (conseillé)

Les conditions :

En Python, les conditions permettent de prendre des décisions dans le programme en fonction de certains critères. Une condition utilise des mots-clés comme if, elif, et else pour exécuter des blocs de code spécifiques selon que des expressions soient vraies ou fausses. Par exemple, if age >= 18: vérifie si age est supérieur ou égal à 18, et exécute le code qui suit uniquement si la condition est vraie. Les conditions rendent le programme interactif et permettent d’adapter son comportement à différentes situations.

age = int(input("Quelle àge avez vous ? :")) #Le int() devant le input sert a convertir la valeur en chiffre entier


if age >= 18: #Si age est supérieur ou égale à 18

    print("Vous êtes majeur !") #Alors envoi dans la console "Vous êtes majeur"

elif age <= 3: #Ajout d'une condition supplémentaires (pas obligatoire). Si age est plus petit ou égale a 3.

    print("Vous êtes trop jeune !")

else: #Sinon

    print("Vous êtes mineur !") #Envoi dans la console "Vous êtes mineur"

Les boucles :

En python il éxiste deux type de boucles des boucles : while et for. Les boucles while servent a intégré une condition dedans, elle contine a s'exécuter tant que la condition est vraie Elle est utile quand on ne sait pas d'avance combien d'itérations seront nécessaires. La boucle for est souvent utilisée pour itérer sur une séquence (comme une liste, une chaîne de caractères, un dictionnaire, ou un intervalle de nombres).

message = "Ceci est une boucle."

count = 0


for i in range(100): #i prendra les valeurs 0, 1, 2, 3, et 4. La valeur dans le range est le nombre de fois que la bouche va être répété

    count += 1 #Ajoute +1 a "count"

    print(f"{count}. { message}") #Envoi dans la console : 1. Ceci est une boucle.



while count < 10:

    print(f"{count}. { message}") #Envoi dans la console : 1. Ceci est une boucle.
